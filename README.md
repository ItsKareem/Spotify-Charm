# Spotify Charm ⋆ ݁ ♫ .

note: I still haven't done the Spotify API code for this. I just shipped it now for the [Arcana](https://arcana.hackclub.com/) deadline.

This Spotify Charm is a palm-sized, fully functional hardware display and controller that uses Spotify, designed with a retro aesthetic inspired by old-school MP3 players. I want to built this irl and use clay and paint to gice it a seashell shap and use actual seashells and use custom jewelery and bead kits to make it an actual charm. I feel like it would be fun to use this instead of my phone. I was inspired by online videos of people making their own creative hardware and physical media projects!

## Features
- Wemos ESP32-C3 Mini development board acting as the main controller
- Rotary Encoder for volume control
- Triple Button Layout ◁ㅤ ❚❚ ㅤ▷ (+ a power switch)
- uses a LiPo Battery Support with full power management circuitry
- Custom 3D-Printed Case featuring a 9cm diameter circular design
